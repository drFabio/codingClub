Uma introdução

Nessa sessão nós vamos iniciar nosso desenvolvimento de html, uma das based das tecnologias Web e o início das habilidades empregáveis para uma carreira fullstack

O HTML foi inventado para marcar HyperTexto, um texto com mais capacidades . Principalmente a capacidade de referenciar outros textos. Ele foi inventado por Tim Bernes Lee em junho de 1993

Hoje em dia HTML é um termo um pouco mais amplo que a linguagem em si, na forma do HTML5 . Um "living standard" que é capitaneada pela WHATWG e auxiliado pela W3C

OK! e na prática?

Na prática o HTML é a base da web, todos os sites que vocês interagem atualmente são consumidos com um documento HTML sendo servido.

Entender HTML nos ajudará também a entender linguagens declarativas

O html como linguagem, é definito por tags e atributos
